# Markdown Notes

> Markdown 是一种轻量级标记语言，可以使用简单的符号编写格式清晰的文档。  

---



## 1. 标题



Markdown 支持六级标题，在文字前添加 `#` 即可。



```markdown

# 一级标题

## 二级标题

### 三级标题

```



建议在 `#` 和标题文字之间保留一个空格。



---


## 3. 换行  

在一行末尾输入两个空格，然后按回车。



```markdown

第一行末尾有两个空格。  

这是第二行。

```

---



## 4. 粗体、斜体和删除线



### 4.1 粗体



```markdown

**这是粗体文字**



__这也是粗体文字__

```



显示效果：



**这是粗体文字**  





### 4.2 斜体



```markdown

*这是斜体文字*



_这也是斜体文字_

```



显示效果：



*这是斜体文字*



### 4.3 粗斜体



```markdown

***这是粗斜体文字***



___这也是粗斜体文字___

```



显示效果：



***这是粗斜体文字***



### 4.4 删除线



```markdown

~~这是被删除的文字~~

```



显示效果：



~~这是被删除的文字~~



删除线属于扩展语法，大多数 Markdown 编辑器和平台都支持。



---



## 5. 无序列表



使用 `-`、`+` 或 `\*` 创建无序列表。



```markdown

- 苹果

- 香蕉

- 橙子

```

* 苹果  
* 香蕉  
* 橙子



建议统一使用 `-`，方便保持文档风格一致。



---



## 6. 多级列表



通过缩进创建嵌套列表，通常使用两个或四个空格。




---



## 8. 任务列表



任务列表常用于待办事项。



```markdown

- [x] 学习标题语法

- [x] 学习列表语法

- [ ] 学习表格语法

- [ ] 完成练习

```



其中：



\- `[x]` 表示已经完成

\- `[ ]` 表示尚未完成


任务列表属于扩展语法，GitHub、GitLab、Typora 等平台通常都支持。



---



## 9. 引用



使用 `>` 创建引用。



```markdown

> 这是一段引用文字。

```



### 9.1 多行引用



```markdown

> 这是引用的第一行。

>

> 这是引用的第二段。

```



### 9.2 嵌套引用



```markdown

> 一级引用

>

>> 二级引用

>>

>>> 三级引用

```


---



## 10. 行内代码



使用一对反引号包裹代码。



```markdown

使用 `print()` 函数输出内容。

```



显示效果：



使用 `print()` 函数输出内容。



如果内容本身包含反引号，可以使用两个反引号包裹：



```markdown

``代码中包含 ` 反引号``

```

## 13. 分隔线



使用三个或更多的 `-`、`\*` 或 `\_` 创建分隔线。



```markdown

---



***



___

```



为了避免与其他语法混淆，建议在分隔线前后各保留一个空行。



---



## 14. 超链接



### 14.1 普通链接



```markdown

[OpenAI 官网](https://openai.com)

```



基本格式：



```text

[链接文字](链接地址)

```



### 14.2 带提示文字的链接



```markdown

[OpenAI 官网](https://openai.com "访问 OpenAI")

```



鼠标悬停在链接上时，可能会显示提示文字。





---



## 16. 图片



图片语法与链接类似，只是在前面增加一个感叹号 `!`。



```markdown

![图片说明](https://example.com/image.png)

```



基本格式：



```text

![替代文字](图片地址)

```



### 16.1 带提示文字的图片



```markdown

![图片说明](https://example.com/image.png "图片标题")

```



### 16.2 使用本地图片



假设 Markdown 文件和 `images` 文件夹位于同一个目录：



```markdown

![项目截图](./images/screenshot.png)

```



上一级目录中的图片：



```markdown

![项目截图](../images/screenshot.png)

```



### 16.3 点击图片跳转



可以将图片语法放进链接语法中：



```markdown

[![图片说明](https://example.com/image.png)](https://example.com)

```

## 18. 转义字符



当需要显示 Markdown 特殊符号本身时，可以在符号前添加反斜杠 `\`。


常见可转义字符：



```text

\  反斜杠

`  反引号

*  星号

_  下划线

{} 花括号

[] 方括号

() 圆括号

#  井号

+  加号

-  减号

.  英文句点

!  感叹号

|  竖线

>  大于号

```

## 26. 高亮文字



部分编辑器支持使用双等号高亮文字。



```markdown

这是普通文字，==这是高亮文字==。

```


==这是高亮文字==


高亮不是标准 Markdown 语法，Obsidian、Typora 等工具可能支持。
